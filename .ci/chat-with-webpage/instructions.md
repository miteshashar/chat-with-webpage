# Instructions

## Backend

```
cd backend
npm install
node server.js
```

## Web Application

```
cd web-app
./run.sh
```

OR

```
cd web-app
npx serve -p 5173
```

## Chrome Extension

1. Open Chrome and navigate to `chrome://extensions/`
2. Enable "Developer mode" in the top right corner
3. Click on "Load unpacked"
4. Select the `chrome-extension` directory
